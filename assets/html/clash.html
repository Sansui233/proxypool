Free Proxies

自动抓取tg频道、订阅地址、公开互联网上的ss、ssr、vmess、trojan节点信息，聚合去重后提供节点列表

Clash

    配置文件

  • Clash配置文件: https://{{ .domain }}/clash/config 一键导入
  • Clash proxy-provider：https://{{ .domain }}/clash/proxies
  • 本地运行时Clash配置文件: http://127.0.0.1:8080/clash/localconfig 一键导入

    • Provider

  • 筛选代理类型：https://{{ .domain }}/clash/proxies?type=ss,ssr,vmess
  • 筛选国家：https://{{ .domain }}/clash/proxies?c=HK,TW,US&nc=JP (c为需要的国家，nc为不需要的国家)
  • 所有节点的Provider(不是都可以用)：https://{{ .domain }}/clash/proxies?type=all
  • 更新方法：
    • Clash for Windows: 更新配置文件即可
    • ClashX: Dashboard中，点击每个策略组右上角的刷新图标
    • ClashXPro: Config → Update External Sources
